react-class-shifter
===================

A React component that facilitates animations by shifting classes around.


```jsx
React.render((
  <ClassShifter>
    <div key="hey">hey there</div>
    <div key="hi">hi there</div>
    <div key="ho">ho there</div>
  </ClassShifter>
), node);
```
